# рЃбрЃљрЃАрЃЎрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў / Task Description Validation Guide

## РюЁ рЃЊрЃўрЃљрЃ«, рЃерЃћрЃЏрЃўрЃФрЃџрЃўрЃљ рЃбрЃљрЃАрЃЎрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ!

**PhoenixExpert** рЃљрЃњрЃћрЃюрЃбрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ:
- РюЁ рЃбрЃљрЃАрЃЎрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў
- РюЁ Confluence-рЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ рЃарЃћрЃљрЃџрЃБрЃарЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
- РюЁ рЃЎрЃЮрЃЊрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ рЃарЃћрЃљрЃџрЃБрЃарЃў implementation-рЃўрЃАрЃЌрЃЋрЃўрЃА
- РюЁ рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ рЃарЃљрЃЏрЃЊрЃћрЃюрЃљрЃЊ рЃерЃћрЃћрЃАрЃљрЃЉрЃљрЃЏрЃћрЃЉрЃљ
- РюЁ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў рЃЊрЃљ рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃў

## рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА / How It Works

### рЃърЃарЃЮрЃфрЃћрЃАрЃў / Process

рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЏрЃЮрЃЏрЃфрЃћрЃЏрЃЌ рЃбрЃљрЃАрЃЎрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљрЃА, рЃЏрЃћ:

1. **рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў** - Confluence-рЃА рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉ MCP tools-рЃўрЃЌ:
   - рЃЋрЃћрЃФрЃћрЃЉ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃА
   - рЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉ business rules рЃЊрЃљ requirements
   - рЃЋрЃърЃЮрЃБрЃџрЃЮрЃЉ API documentation рЃЊрЃљ specifications

2. **рЃЏрЃћрЃЮрЃарЃћ рЃарЃўрЃњрЃерЃў** - Phoenix рЃЎрЃЮрЃЊрЃЉрЃљрЃќрЃљрЃА рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉ:
   - рЃЋрЃћрЃФрЃћрЃЉ controllers, services, models
   - рЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉ validation rules
   - рЃЋрЃърЃЮрЃБрЃџрЃЮрЃЉ endpoints рЃЊрЃљ business logic
   - рЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉ рЃарЃћрЃљрЃџрЃБрЃа implementation-рЃА

3. **рЃерЃћрЃЏрЃЊрЃћрЃњ** - рЃЋрЃљрЃЎрЃћрЃЌрЃћрЃЉ рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљрЃА:
   - рЃарЃљрЃЏрЃЊрЃћрЃюрЃљрЃЊ рЃерЃћрЃћрЃАрЃљрЃЉрЃљрЃЏрЃћрЃЉрЃљ рЃбрЃљрЃАрЃЎрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ Confluence-рЃА
   - рЃарЃљрЃЏрЃЊрЃћрЃюрЃљрЃЊ рЃерЃћрЃћрЃАрЃљрЃЉрЃљрЃЏрЃћрЃЉрЃљ рЃбрЃљрЃАрЃЎрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ рЃЎрЃЮрЃЊрЃА
   - рЃарЃљ рЃљрЃарЃўрЃА рЃАрЃгрЃЮрЃарЃў рЃЊрЃљ рЃарЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃЊрЃћрЃА

4. **рЃЊрЃљрЃАрЃљрЃАрЃарЃБрЃџрЃА** - рЃЋрЃљрЃФрЃџрЃћрЃЋ рЃЊрЃћрЃбрЃљрЃџрЃБрЃа рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљрЃА:
   - РюЁ рЃарЃљ рЃљрЃарЃўрЃА рЃАрЃгрЃЮрЃарЃў
   - Рџа№ИЈ рЃарЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃЊрЃћрЃА
   - РЮї рЃарЃљ рЃљрЃа рЃерЃћрЃћрЃАрЃљрЃЉрЃљрЃЏрЃћрЃЉрЃљ
   - ­ЪЊЮ рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃў

## рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў / Examples

### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 1: рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃбрЃљрЃАрЃЎрЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ

**рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃбрЃљрЃАрЃЎрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ:**
```
"рЃЊрЃљрЃЋрЃљрЃЏрЃљрЃбрЃЮрЃЌ validation customer-рЃўрЃА email-рЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏ рЃўрЃДрЃЮрЃА unique"
```

**рЃЕрЃћрЃЏрЃў рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ:**
1. РюЁ **Confluence рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ**: рЃЋрЃћрЃФрЃћрЃЉ customer validation rules
2. РюЁ **рЃЎрЃЮрЃЊрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ**: рЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉ Customer entity рЃЊрЃљ validation annotations
3. РюЁ **рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ**: рЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉ рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃБрЃЎрЃЋрЃћ unique constraint
4. ­ЪЊі **рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ**: 
   - рЃАрЃгрЃЮрЃарЃўрЃљ рЃЌрЃБ рЃљрЃарЃљ requirement-рЃў
   - рЃерЃћрЃћрЃАрЃљрЃЉрЃљрЃЏрЃћрЃЉрЃљ рЃЌрЃБ рЃљрЃарЃљ рЃЎрЃЮрЃЊрЃА
   - рЃарЃљ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЎрЃћрЃЌрЃЊрЃћрЃА

### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 2: рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў рЃбрЃљрЃАрЃЎрЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ

**рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃбрЃљрЃАрЃЎрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ:**
```
"рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ рЃљрЃ«рЃљрЃџрЃў endpoint /api/billing/calculate рЃарЃЮрЃЏрЃћрЃџрЃўрЃф:
- рЃўрЃдрЃћрЃЉрЃА customerId рЃЊрЃљ period
- рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА billing amount
- рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА permissions
- рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ request-рЃА"
```

**рЃЕрЃћрЃЏрЃў рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ:**
1. РюЁ **Confluence**: рЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉ billing API documentation
2. РюЁ **рЃЎрЃЮрЃЊрЃў**: рЃЋрЃћрЃФрЃћрЃЉ BillingController, BillingService, existing endpoints
3. РюЁ **рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ**: 
   - рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃЌрЃБ рЃљрЃарЃљ рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў endpoint
   - рЃарЃљ permissions рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ
   - рЃарЃЮрЃњрЃЮрЃа рЃџрЃЮрЃњрЃўрЃарЃЊрЃћрЃЉрЃљ requests
4. ­ЪЊі **рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ**:
   - рЃарЃљ рЃљрЃарЃўрЃА рЃАрЃгрЃЮрЃарЃў requirement-рЃћрЃЉрЃерЃў
   - рЃарЃљ рЃљрЃа рЃерЃћрЃћрЃАрЃљрЃЉрЃљрЃЏрЃћрЃЉрЃљ рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃЎрЃЮрЃЊрЃА
   - рЃарЃљ рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃћрЃЏрЃљрЃбрЃЮрЃА
   - рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃў implementation-рЃўрЃАрЃЌрЃЋрЃўрЃА

## рЃарЃљ рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљрЃА рЃЏрЃўрЃЋрЃўрЃдрЃћрЃЉрЃЌ / What You'll Get

### ­ЪЊі рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

```markdown
## рЃбрЃљрЃАрЃЎрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ

### РюЁ Confluence рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃЮрЃЉрЃљ
- [рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў рЃарЃљ рЃўрЃърЃЮрЃЋрЃљ Confluence-рЃерЃў]
- [рЃарЃљрЃЏрЃЊрЃћрЃюрЃљрЃЊ рЃерЃћрЃћрЃАрЃљрЃЉрЃљрЃЏрЃћрЃЉрЃљ]

### РюЁ рЃЎрЃЮрЃЊрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃЮрЃЉрЃљ  
- [рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў рЃарЃљ рЃўрЃърЃЮрЃЋрЃљ рЃЎрЃЮрЃЊрЃерЃў]
- [рЃарЃљрЃЏрЃЊрЃћрЃюрЃљрЃЊ рЃерЃћрЃћрЃАрЃљрЃЉрЃљрЃЏрЃћрЃЉрЃљ]

### ­ЪЊІ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў
- РюЁ рЃАрЃгрЃЮрЃарЃў рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃў
- Рџа№ИЈ рЃњрЃљрЃАрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃћрЃџрЃў рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃў
- РЮї рЃерЃћрЃБрЃАрЃљрЃЉрЃљрЃЏрЃЮ рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃў

### ­ЪњА рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃў
- [рЃарЃљ рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃЮрЃА]
- [рЃарЃљ рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃћрЃЏрЃљрЃбрЃЮрЃА]
```

## рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ / How to Use

### рЃЏрЃћрЃЌрЃЮрЃЊрЃў 1: Cursor Chat-рЃерЃў рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа

рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃЊрЃљрЃгрЃћрЃарЃћрЃЌ:

```
рЃерЃћрЃљрЃцрЃљрЃАрЃћ рЃћрЃА рЃбрЃљрЃАрЃЎрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ:
[рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃбрЃљрЃАрЃЎрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ]

рЃЊрЃљ рЃерЃћрЃљрЃЊрЃљрЃарЃћ Confluence-рЃАрЃљ рЃЊрЃљ рЃЎрЃЮрЃЊрЃА.
```

**рЃЏрЃћ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ:**
1. рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЉ PhoenixExpert-рЃА
2. рЃљрЃЏрЃЮрЃЋрЃгрЃЏрЃћрЃЉ Confluence-рЃА MCP tools-рЃўрЃЌ
3. рЃљрЃЏрЃЮрЃЋрЃгрЃЏрЃћрЃЉ Phoenix рЃЎрЃЮрЃЊрЃЉрЃљрЃќрЃљрЃА
4. рЃњрЃљрЃњрЃфрЃћрЃЏрЃЌ рЃЊрЃћрЃбрЃљрЃџрЃБрЃа рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљрЃА

### рЃЏрЃћрЃЌрЃЮрЃЊрЃў 2: рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃЎрЃБрЃарЃў рЃЎрЃўрЃЌрЃ«рЃЋрЃљ
